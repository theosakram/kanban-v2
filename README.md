# kanban-v2 (WORK IN PROGRESS)

This is a personal fullstack project

## Technologies Used

## 1. Backend

- Rust
- Actix Web
- Shuttle
- PostgreSQL (No ORM)

### BE Features

- Auth
- CRUD
- RBAC (Role-Based Access Control)

## 2. Frontend

- Nexjs
- Typescript

### FE Features

- Drag n Drop
- RBAC (Role-Based Access Control)
- Client side cache

## 3. Caching

Cached with actix-cache
